# 🏥 Our Health - Comprehensive Health & Nutrition Web Application

## 📋 Project Overview

**Our Health** is a comprehensive web-based health and nutrition application designed to empower users to take control of their wellbeing. The platform provides personalized diet plans, detailed food nutritional information, exercise routines with video demonstrations, and regional dietary guidance tailored to different Indian cuisines and cultural preferences.

**Key Tagline:** "Fuel your body, fuel your life - customized diet plans just for you."

---

## 🎯 Core Features

### 1. **Diet Planning**
   - **Weight Loss Plans** - Customized meal plans for weight reduction
   - **Weight Gain Plans** - Nutrition strategies for healthy weight gain
   - **Weight Maintenance Plans** - Balanced diet plans to maintain current weight
   - **Regional Diet Guides** - Specialized meal plans based on Indian regional cuisines:
     - East Indian (Veg & Non-Veg)
     - North Indian (Veg & Non-Veg)
     - South Indian (Veg & Non-Veg)
     - West Indian (Veg & Non-Veg)
     - International options (Veg & Non-Veg)
   - **Balanced Diet Charts** - Visual representations of balanced diet composition

### 2. **Food & Nutrients Database**
   - Comprehensive nutrient information for 40+ food items
   - Categorized by nutrient type:
     - **Proteins** (Eggs, Almonds, Chicken, Fish, Milk, Soybeans, Peanuts, Bread, Seeds, Cheese, Peanut Butter)
     - **Carbohydrates** (Sweet Potato, Beetroot, Corn, Quinoa, Brown Rice, Oats, Bananas, Apple, Mango, Dates, Raisins, Beans, Wheat)
     - **Fats** (Fish, Eggs, Almonds, Peanuts, Chia Seeds, Dark Chocolate, Avocado, Flaxseeds, Olives, Tofu, Walnuts, Soymilk)
     - **Vitamins** (Fruits, Fish, Green Leafy Vegetables, Broccoli, Mushrooms, Bell Peppers, Peas, Winter Squash, Carrots, Asparagus, Eggs, Avocado)
     - **Minerals** (Nuts/Seeds, Shellfish, Cabbage, Meats, Beans, Cocoa, Berries, Sardines, Spirulina, Starchy Vegetables, Eggs, Avocado)
     - **Zinc** (Meats, Shellfish, Legumes, Seeds, Nuts, Cocoa, Poultry)
     - **Iron** (Red Meats, Poultry, Fish, Legumes, Leafy Greens, Fortified Cereals)
   - Detailed nutritional values in grams and ounces
   - High-quality food images (AVIF format for optimization)

### 3. **Exercise Routines**
   - **Customizable Workouts** - Select fitness goal and difficulty level
   - **Fitness Goals:**
     - Weight Loss
     - Muscle Gain
     - Weight Maintenance
   - **Fitness Levels:**
     - Beginner
     - Intermediate
     - Advanced
   - **Video Demonstrations** - MP4 videos showing proper exercise form:
     - Jumping Jacks
     - High Knees
     - Squats (Regular & Weighted)
     - Push-ups
     - Plank
     - Pull-ups
     - Dips
     - Mountain Climbers
     - Burpees
     - Deadlifts
     - Bench Press
     - Overhead Press
     - Glute Bridges

### 4. **Assessment Questionnaire**
   - Interactive quiz to determine personalized recommendations (Swipper)
   - Factors considered:
     - Current weight status
     - Fitness goals
     - Dietary preferences (Veg/Non-Veg)
     - Regional food preferences

## 🚀 How to Use

### **For End Users:**

1. **Start at Home** - Visit `index.html` to see the landing page
2. **Take Assessment** - Click "Get Started" to access the interactive quiz
3. **Explore Food Database** - Navigate to "FOOD" to learn about nutrients in different foods
4. **Choose Diet Plan** - Go to "DIET" section to select appropriate meal plans based on:
   - Weight loss/gain/maintenance goals
   - Dietary preference (Veg/Non-Veg)
   - Regional cuisine preference
5. **Get Exercise Routines** - Visit "EXERCISES" to:
   - Select fitness goal
   - Choose fitness level
   - Watch demonstration videos
6. **Check Regional Guides** - View balanced diet charts specific to your region

### **For Developers:**

1. **Local Setup:**
   ```bash
   git clone <repository>
   cd "Our Health"
   ```

2. **Run Locally:**
   - Open any `.html` file in a web browser
   - Or use a local server:
     ```bash
     python -m http.server 8000
     ```

3. **Customize Content:**
   - Edit HTML files for content changes
   - Modify CSS files for styling updates
   - Update JavaScript for functionality changes

4. **Add New Features:**
   - Follow existing naming conventions
   - Maintain consistent folder structure
   - Test across different browsers
